# Artisan Square Catalog — Overview

## Purpose

The artisan-square catalog describes Artisan Square facility configurations — multi-trade
production buildings that provide individual rented studios to artisan practitioners. Each
entry is a building design evaluated under market/coop/civic lenses as if the building
developer/operator were the "practitioner." The operator provides space, infrastructure,
and compliance umbrella; tenant craft decisions, customer relationships, and brand belong
entirely to each tenant. 15 configurations are planned, spanning small/medium/large
formats and rural/suburban/urban contexts. Plan K will fully author each entry.

### Group A — Medium/Large Combined Campus (5 entries)

Combined-campus designs hosting both Type A (heavy-trades) and Type B (light-trades)
wings under one roof. These represent the core Artisan Square model.

| square-002 | Medium purpose-built, suburban |
| square-003 | Large combined campus, urban |

| square-005 | Large destination campus with retail gallery |

### Group B — Type B Light-trades Only (4 entries)

Single-wing designs serving light trades only (baking, fiber arts, leather, ceramics).
No Type A heavy-trade infrastructure; lower build-out cost and simpler permitting.

| ID | One-line description |
|---|---|
| square-006 | Small Type B, mixed-use district |
| square-007 | Medium Type B, street-level retail |
| square-008 | Medium Type B with commissary + food hall |
| square-009 | Small Type B, small-town adaptation |

### Group C — Type A Heavy-trades Only (3 entries)

Single-wing designs serving heavy trades only (smithing, metalwork, pottery/kilns).
Full 3-phase infrastructure and industrial ventilation; no food-grade commissary required.

| ID | One-line description |
|---|---|
| square-010 | Small Type A, light-industrial district |
| square-011 | Medium Type A with shared metalwork shop |
| square-012 | Small Type A, small-town with civic forge anchor |

### Group D — Small-Town Lite (3 entries)

Scaled-down configurations for towns of 2,000–15,000 population — the CERES sweet spot
for civic co-investment and thin practitioner markets.

| ID | One-line description |
|---|---|
| square-013 | Small combined, small town (pop 2k–5k) |
| square-014 | Type B only, small town (pop 5k–15k) |
| square-015 | Minimum-viable single-trade, village |
