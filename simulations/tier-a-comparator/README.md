# simulations/tier-a-comparator/

Tier A: deterministic scenario comparator. Evaluates every catalog entry
against all 9 contexts (3 scales × 3 economic lenses) using closed-form math.
Produces the primary evaluation matrix. Entry point: `model.py` (or
`model.ipynb`). Results stored in `results/`.

Populated by Plan D — scenario comparator is the first simulation deliverable.
